## Description

**Mabell** is the client-side application for a music streaming service.

The application allows users to search, listen to music, create playlists, and receive personalized recommendations based on their preferences. It is optimized for performance and offers a seamless user experience across various devices.

This repository contains the source code for the frontend, leveraging the power of modern frameworks and tools to ensure scalability, maintainability, and flexibility. The code is organized using a **modular architecture**, where the project is divided into Nuxt modules. This approach allows for better separation of concerns, easier maintainability, and the ability to scale the application by adding new features in isolated, reusable modules.

## Technologies Used

- **Nuxt 3**: A powerful framework built on top of Vue.js to enable server-side rendering, static site generation, and API integration.
- **TypeScript**: For static typing, making the development process more predictable and reliable.
- **Pinia**: A state management library for Vue 3, used for managing global app state with simplicity and flexibility.
- **SCSS**: For styling the application with modular, reusable, and maintainable styles.
- **VTL (Vue Template Language)**: A powerful templating language to create dynamic and flexible UI components.
- **Docker**: Containerized environment for easy setup, deployment, and scalability of the application.
